🔣 Знаки в Python

Умножение = звёздочка *
Деление = слэш /
Остаток от деления = %
Степень = ** (например 2**3 = 8)

💻 Примеры на Python
# Простые примеры print(5 + 3) # = 8 print(10 - 4) # = 6 print(3 * 4) # = 12 print(15 / 3) # = 5.0 # С переменными яблоки = 8 съел = 3 осталось = яблоки - съел print(f"Осталось яблок: {осталось}")
📺 Результат:
8 6 12 5.0 Осталось яблок: 5
🧠 Задачи в Python
🍕 Задача 1: Пицца

В пицце 8 кусков. Вас 3 человека. Сколько кусков достанется каждому?

кусков = 8 человек = 3 каждому = кусков // человек # // = деление без остатка остаток = кусков % человек # % = остаток print(f"Каждому: {каждому} кусков") print(f"Лишних кусков: {остаток}")
🚀 Задача 2: Скорость ракеты

Ракета летит со скоростью 7900 метров в секунду. Сколько километров она пролетит за 1 минуту?

скорость_мс = 7900 # метров в секунду секунд = 60 # 1 минута метров = скорость_мс * секунд километров = метров / 1000 print(f"Ракета летит {километров} км за минуту!")
📐 Порядок действий — как в математике!
Python умеет скобки!

Правила те же, что в математике: сначала скобки, потом * и /, потом + и -

print(2 + 3 * 4) # = 14 (сначала умножение!) print((2 + 3) * 4) # = 20 (сначала скобки!)
