Команда input() останавливает программу и ждёт, когда пользователь введёт что-то с клавиатуры!

# Спрашиваем имя имя = input("Как тебя зовут? ") print(f"Привет, {имя}!")
⚠️ Важно! Числа нужно преобразовать

input() всегда возвращает текст (строку). Чтобы считать с числами, нужно написать int()!

текст = input("Введи число: ") # это текст "5" число = int(текст) # это число 5 print(число + 10) # = 15
🧮 Пишем калькулятор!
📋 Наш план:

1. Спрашиваем первое число
2. Спрашиваем второе число
3. Считаем все 4 действия
4. Показываем результаты!

# 🧮 МОЙ КАЛЬКУЛЯТОР print("=== 🧮 Мой Калькулятор ===") print() # Спрашиваем числа число1 = int(input("Введи первое число: ")) число2 = int(input("Введи второе число: ")) # Считаем сумма = число1 + число2 разность = число1 - число2 произв = число1 * число2 # Показываем результаты print() print(f"➕ {число1} + {число2} = {сумма}") print(f"➖ {число1} - {число2} = {разность}") print(f"✖️ {число1} × {число2} = {произв}") # Деление — проверяем что не делим на 0! if число2 != 0: частное = число1 / число2 print(f"➗ {число1} ÷ {число2} = {частное}") else: print("➗ На ноль делить нельзя! 😅") print() print("Спасибо за использование калькулятора! 🏆")

➕ 12 + 4 = 16

🆕 Что узнали нового
  • input() — спрашивает у пользователя данные
  • int() — превращает текст в число
  • if / else — условие: «если... то... иначе...»
  • != — означает «не равно»
  • print() без аргументов — пустая строка
