💻 Первая команда — print()
📢 print — значит «напечатать»

Команда print() показывает текст на экране. Текст нужно взять в кавычки!

# Моя первая программа на Python! print("Привет, мир!") print("Меня зовут Тимур") print("Я учусь программировать!")
📺 Результат на экране:
Привет, мир! Меня зовут Тимур Я учусь программировать!
📏 Важные правила Python
  • Код пишется по одной команде на строку
  • Python чувствителен к регистру: Print ≠ print
  • Текст в кавычках — это строка (String)
  • Строка начинающаяся с # — это комментарий (компьютер игнорирует)
  • Числа пишутся без кавычек: 42, 3.14
