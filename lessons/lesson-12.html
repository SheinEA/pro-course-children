📦 Что такое переменная?
🗃️ Переменная — как коробка с ярлычком

Представь коробку, на которой написано «яблоки». Внутри — 5 яблок. Переменная — это такая коробка! У неё есть имя (ярлычок) и значение (то что внутри).

📦
имя = "Тимур"
📦
возраст = 7
💻 Создаём переменные в Python
# Создаём переменные имя = "Тимур" возраст = 7 любимый_цвет = "синий" оценка = 5 # Используем переменные print(имя) print(возраст) print(любимый_цвет)
📺 Результат:
Тимур 7 синий
🗂️ Виды данных в переменных
📝 Строка (str) — текст в кавычках
имя = "Тимур" # текст
🔢 Целое число (int) — без точки
возраст = 7 # целое число
📊 Дробное число (float) — с точкой
рост = 1.25 # число с дробью
✅ Правда/Ложь (bool) — True или False
люблю_мороженое = True # да!
✨ Текст + переменные вместе
имя = "Тимур" возраст = 7 # Способ 1: через запятую print("Привет, меня зовут", имя) print("Мне", возраст, "лет") # Способ 2: f-строка (модный способ!) print(f"Привет! Я {имя}, мне {возраст} лет!")
📺 Результат:
Привет, меня зовут Тимур Мне 7 лет Привет! Я Тимур, мне 7 лет!
🌟 Интересный факт!

В компьютере все переменные хранятся в оперативной памяти (RAM) — той самой, о которой мы говорили в уроке 2! Когда программа закрывается — переменные исчезают. Поэтому важные данные сохраняют в файлы! 💾
