🔧 Что такое IDE?
🏗️ IDE = Мастерская программиста

IDE (Integrated Development Environment) — это специальная программа для написания кода. Как мастерская у папы — там есть все инструменты в одном месте!

✨ Подсветка кода

Разные слова красятся в разные цвета — так легче читать!

🔮 Автодополнение

IDE подсказывает что написать дальше — как умная клавиатура!

🐛 Поиск ошибок

IDE показывает где ошибка — подчёркивает красным!

▶️ Запуск кода

Нажал кнопку — программа запустилась прямо здесь!

⚙️ Как установить
Шаг 1
Скачай Python на python.org → нажми «Download»
Шаг 2
При установке Python — поставь галочку ✅ «Add to PATH»!
Шаг 3
Скачай VS Code на code.visualstudio.com
Шаг 4
В VS Code установи расширение «Python» (иконка пазла)
Шаг 5
Создай файл hello.py и напиши print("Привет!") → запусти!
🌟 Интересный факт!

VS Code скачивают более 14 миллионов раз в месяц! По опросам разработчиков, это самый любимый редактор в мире уже несколько лет подряд. Твой папа тоже наверняка им пользуется! Теперь у вас будет один любимый инструмент! 👨‍💻👦
