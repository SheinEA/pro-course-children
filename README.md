# 🚀 IT-Курс для Героев — Программирование для детей

Курс для погружения ребёнка (7 лет) в IT и программирование.

## 📋 О курсе

- **Возраст:** 7 лет
- **Длительность урока:** 20 минут
- **Периодичность:** 1 раз в неделю
- **Всего уроков:** 18
- **Формат:** Красочные веб-презентации с навигацией

## 🎯 Цели курса

По завершению курса ребёнок:
- ✅ Умеет печатать (слепой метод)
- ✅ Знает устройство компьютера и ОС
- ✅ Умеет пользоваться IDE (VS Code)
- ✅ Пишет простой код на Python (арифметика, калькулятор)
- ✅ Создаёт игры в Scratch
- ✅ Использует AI-инструменты для создания игр

## 📚 Программа курса

### Блок 1: Основы компьютера (Уроки 1-4) ✅ Пройдено
1. Устройства ввода и вывода
2. Системный блок и его компоненты
3. Операционные системы (Windows, macOS, Linux, Android)
4. Рабочий стол Windows

### Блок 2: Интернет и клавиатура (Уроки 5-7)
5. Интернет и браузеры
6. Знакомство с клавиатурой
7. Учимся печатать быстро

### Блок 3: Scratch — визуальное программирование (Уроки 8-10)
8. Scratch — что такое программирование?
9. Scratch — первая анимация
10. Scratch — первая игра

### Блок 4: Python — настоящий код (Уроки 11-14)
11. Python — что такое код?
12. Python — числа и переменные
13. Python — арифметика
14. Python — пишем калькулятор

### Блок 5: IDE и Искусственный Интеллект (Уроки 15-18)
15. IDE — редактор кода (VS Code)
16. Что такое Искусственный Интеллект?
17. AI создаёт игры
18. Финальный проект — Я программист!

## 🌐 Просмотр курса

Откройте файл `index.html` в браузере для просмотра главной страницы курса.

## 🛠️ Технологии

- HTML5 + CSS3
- Google Fonts (Nunito)
- Без зависимостей — работает offline

## 👨‍👦 Учитель и ученик

**Учитель:** Отец, профессиональный программист
**Ученик:** Сын, 7 лет, любит рисовать

Каждый урок содержит домашнее задание с рисованием — специально для ребёнка, который любит рисовать!
